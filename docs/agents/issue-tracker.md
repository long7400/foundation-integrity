# Issue tracker

This repository's issue tracker is GitHub Issues for
`long7400/foundation-integrity`, evidenced by the `origin` remote. Use `gh issue`
commands for issue reads and writes when an external issue is the canonical source.

The Foundation Integrity gate remains the owner of foundation receipts, ADRs, and
acceptance decisions. A GitHub issue is an input or coordination record; it does not
override a receipt, the task graph, or root acceptance.

Pull requests are not automatically triage items. Add that behavior only as an
explicit repository decision.
